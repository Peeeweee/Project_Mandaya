# LOGGED OUT — Restoring the Deleted Repository of CARAGA

A cinematic, interactive React presentation exploring the metaphor of cultural erasure and restoration through the lens of a deleted Git repository.

## Description
This project reimagines the history, erasure, and enduring resilience of the Indigenous Peoples of the CARAGA region (Mindanao, Philippines). Designed as an aesthetic, terminal-inspired slide deck, it walks the audience through "commit history," file restorations, and cultural data ranging from the Mandaya *Dagmay* textiles to Manobo and Higaonon traditions.

## Navigation & Controls
The presentation is fully keyboard-driven for a seamless presenter experience:
* <kbd>Space</kbd> or <kbd>→</kbd> or <kbd>↓</kbd> — Next slide / Trigger next animation
* <kbd>←</kbd> or <kbd>↑</kbd> — Previous slide
* <kbd>F</kbd> — Toggle Fullscreen
* <kbd>M</kbd> — Toggle Sound / Mute
* <kbd>Home</kbd> — Jump to the first slide
* <kbd>End</kbd> — Jump to the final slide
* <kbd>1</kbd> - <kbd>9</kbd> — Jump to specific slides directly

*(Note: Audio is muted by default and must be enabled by the user or via the 'M' key once the presentation begins.)*

## Tech Stack
* **React 19** (Functional Components + Hooks)
* **Vite** (Build Tool)
* **Tailwind CSS v4** (Utility Classes)
* **Web Audio API** (Procedural Sound Generation, nil external audio files)
* **HTML5 Canvas** (Dynamic Dagmay pattern rendering)
